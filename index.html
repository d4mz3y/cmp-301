INTRODUCTION

PHP is an acronym for "PHP: Hypertext Preprocessor". It is a very widely-used, open source scripting language. PHP works by using scripts which are executed on a server. PHP is especially suited for web development and can be embedded into HTML. Its widely used and is used in popular sites. It is a loosely typed language i.e. you don't have to specify the datatype when you are declaring or executing variables in the code. The PHP reference implementation is now produced by The PHP Group.PHP originally stood for Personal Home Page, but it now stands for the recursive initialism PHP: Hypertext Preprocessor.
